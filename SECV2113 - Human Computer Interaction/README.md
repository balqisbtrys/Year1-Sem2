This is a foundational course in HCI that introduces core theories, design processes, and applied practices in user experience (UX) design. It emphasizes both conceptual understanding and practical application. This course explores an evaluation-centered UX lifecycle, emphasizing usability, usefulness, and emotional impact as integral components of user experience. 

### Continuous Assessment
1. Assignment (Group)      - 5%
2. Project: Part 1         - 5%
3. Project: Part 2         - 10%
